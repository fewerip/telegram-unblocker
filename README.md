# Telegram Unblocker (Обход замедления Telegram)

[English Version](README_EN.md)

Этот инструмент помогает обойти замедление Telegram Desktop со стороны провайдера.

## Проблема
РКН замедляет трафик Telegram Desktop (MTProto), из-за чего файлы качаются медленно. Обычные SOCKS5 прокси тоже замедляются, так как их "рукопожатие" легко узнаваемо.

## Решение
Эта программа создает локальный прокси SOCKS5, который соединяется с вашим внешним SOCKS5 прокси. Главная фишка — она **фрагментирует (дробит)** пакеты при подключении. Это сбивает с толку DPI провайдера (похоже на принцип работы Zapret/GoodbyeDPI), и скорость восстанавливается.

## Компоненты
-   `Manage.bat`: Главное меню управления (Настройка, Установка, Удаление).
-   `bin/TelegramUnblocker.exe`: Сам движок прокси.
-   `bin/config.json`: Файл с настройками.

## Как пользоваться

### 1. Что нужно
-   Рабочий **Внешний SOCKS5 Прокси** (IP, Порт, Логин, Пароль).
-   Windows 10/11.

### 2. Настройка (Manage.bat)
1.  Запустите `Manage.bat`.
2.  Выберите **пункт 1 (Configure Proxy)**.
    -   **Local Port**: Порт, на котором будет работать программа (по умолчанию `10805`).
    -   **Remote Proxy IP/Port**: Адрес и порт вашего внешнего прокси.
    -   **Username/Password**: Логин и пароль (оставьте пустым, если не нужны).

### 3. Запуск
-   **Тестовый режим**: Выберите **пункт 2 (Test Run)**. Откроется окно с логами. Если ошибок нет — всё работает.
-   **Установка службы**: Выберите **пункт 3 (Install Service)**. Программа установится как Служба Windows (`TelegramUnblocker`) и будет запускаться сама при включении компьютера. Окон видно не будет.
-   **Удаление**: Выберите **пункт 4**, чтобы удалить службу.

### 4. Настройка Telegram Desktop
Когда программа запущена, настройте сам Telegram:

1.  Откройте Telegram -> Настройки.
2.  Перейдите в **Данные и память** (Data and Storage).
3.  Прокрутите вниз до **Тип соединения** (Connection Type) -> Настройки прокси.
4.  Нажмите **Добавить прокси**.
5.  Выберите **SOCKS5**.
6.  **Хост (Socket Address)**: `127.0.0.1`
7.  **Порт**: `10805` (или тот, который вы указали в настройках).
8.  **Логин/Пароль**: Оставьте пустыми (авторизацией занимается наша программа).
9.  Нажмите **Сохранить**.

Значок 'щита' должен показать галочку/пинг. Теперь файлы будут качаться быстро.

## Сборка из исходников

Если вы хотите собрать EXE самостоятельно:
1.  Установите Python.
2.  Установите библиотеки: `pip install pyinstaller pywin32`
3.  Запустите `build.bat` (в папке `bin` появится EXE файл).
